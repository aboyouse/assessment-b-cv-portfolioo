# Assessment B CV Portfolio

Name: Ahmed Aboyousef

## Project Description

This project is a dynamic CV portfolio website created for Assessment B. It is based on my previous CV portfolio from Assessment A, but it has been improved by using JavaScript and JSON.

The website includes a homepage and a CV page. The CV information is stored in `data.json` and loaded into the website using `app.js`.

## Files Included

- `index.html` - the homepage of the portfolio
- `cv.html` - the full CV page
- `style.css` - the styling file used for both pages
- `app.js` - the JavaScript file that loads the JSON data and controls the interactive features
- `data.json` - the file that stores the CV content
- `AhmedAboyousef_bash.sh` - Bash script used to count new learning comments
- `AhmedAboyousef_bash_report.txt` - text file created by the Bash script
- `bash_screenshot.png` - screenshot of the Bash script output

## How to Run Locally

1. Download or clone the project folder.
2. Open the folder in Visual Studio Code.
3. Install the Live Server extension if it is not already installed.
4. Open `index.html`.
5. Right-click and choose `Open with Live Server`.

## AI Tools Used

ChatGPT was used to help with the website development process. It helped with planning the structure, converting the old CV project into an Assessment B version, creating the JSON structure, and explaining the JavaScript code.

## Interactive Features

The website includes:

- Dark Mode button
- Show / Hide buttons for CV sections
- Dynamic loading of CV content from `data.json`
